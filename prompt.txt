أنا عايزك تنشئلي مشروع Cloudflare Worker باستخدام Hono framework، يكون عبارة عن سيرفر جاهز فيه الآتي:

📁 ملفات المشروع المطلوبة:
index.ts: ملف السيرفر الرئيسي، يحتوي على كود HTML مدمج داخل الملف نفسه (inline) + كل المسارات (routes) الخاصة بالتسجيل وتسجيل الدخول.

package.json, tsconfig.json, wrangler.toml: ملفات الإعدادات علشان أقدر أرفعه على Cloudflare Workers مباشرة بدون أي CLI.

المشروع يكون مضغوط في ملف .zip واحد فيه كل الملفات دي، علشان أرفعه على GitHub وأوصله بـ Cloudflare من الواجهة فقط بدون Terminal.

🧠 الوظائف اللي السيرفر لازم يعملها:
🖥️ 1. واجهة HTML مدمجة (inline) في index.ts:
واجهة احترافية بسيطة (تصميم نضيف ومنظم) فيها:

نموذج تسجيل مستخدم جديد (Sign Up) بـ email + password.

نموذج تسجيل دخول (Login) بنفس البيانات.

بعد تسجيل الدخول أو التسجيل، يظهر للمستخدم رسالة ترحيب باسمه أو بريده الإلكتروني.

🔌 2. Routes (مسارات) API داخل السيرفر باستخدام Hono:
POST /signup: يستقبل JSON فيه email وpassword، ويسجلهم في قاعدة بيانات D1 (تكون مربوطة عن طريق wrangler.toml).

POST /login: يتحقق من وجود المستخدم وكلمة السر، ويرجع success أو failure.

GET /: يعرض صفحة الـ HTML.

GET /users: مسار خاص بـ admin فقط لعرض كل المستخدمين (لو حبيت تضيف JWT أو Authorization يكون ممتاز).

🔐 الأمان:
كلمات المرور لازم تتخزن مشفّرة باستخدام bcrypt أو مكتبة بديلة في JavaScript.

التحقق من البريد وكلمة المرور لازم يتم بدقة في تسجيل الدخول.

🛠️ متطلبات تنفيذ المشروع:
يكون ملف index.ts جاهز للنشر فورًا.

قاعدة البيانات D1 تكون معرفة في wrangler.toml بالطريقة الصحيحة، ومربوطة بـ binding = "DB".

المشروع يشتغل بدون أي CLI أو أوامر Terminal.
